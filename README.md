# EMS
Employee Management System

Project Description:
Employee Management System is an administrative module to register newly joined employee,to manage existing employees as per the oraganization standard.It has feature to add/remove departments/project accounts and manage projects in the department by mapping to employee as per Designation.It also create/manage role/responsibilities to the employee as per per project needs and organization demand.
It is basically a web application being developed considering IT industries into account but it will work for any organization.

Features: All new technologies are being used and responsive design with zero defect is primary aim.

Technologies/Skills:JSP,HTML,CSS,Core Java,JQuery,JQuery Ajax.
Framework:Spring MVC       version: 4.3.6
ORM Tool: Hibernate        version: 5.2.8
Database: SQL,HQL          
Database Tool: PostgreSQL  version: 9.4
Server: Tomcat version 9.0
JDK : 1.8
